# Rakshit Tiwari Portfolio

Static portfolio website ready for GitHub + Netlify.

## Publish on Netlify

1. Upload this folder to a GitHub repository.
2. In Netlify, choose **Add new site** and connect that repository.
3. Use these settings:
   - Build command: leave blank
   - Publish directory: `.`
4. Deploy.

If the repository contains other folders, either upload only this folder to GitHub or set the publish directory to `outputs/portfolio-site`.
